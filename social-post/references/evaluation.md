# 貼文評估框架

本檔用於 Phase 2 發完後評估表現、以及使用者問「這篇好不好」時的判斷依據。

## 🧠 FB 2026 演算法訊號權重（最重要）

Meta 2024-11 推「Unconnected Reach」指標。**讚幾乎無權重**，第 1 小時內的三訊號決定是否跨粉絲圈：

| 信號 | 權重 | 說明 |
|---|---|---|
| **分享** | **20× 讚** | 最強廣推 trigger |
| **留言** | **5× 讚** | 次強，深度留言 > 短留言 |
| **Dwell time** | 隱形王牌 | 「查看更多」點擊 + 圖放大 + revisit |
| 讚 | 幾乎 0 | 絕對值無評估意義 |

**判斷順序**：分享 > 留言 > dwell > 讚。

## 🎯 4 指標評估法（取代絕對讚數判斷）

**不要看讚絕對數**。看這四個：

1. **連結點擊率** > 1% = 鐵粉真的行動（點 GitHub / Line 等）
2. **追蹤者比例** > 50% = 演算法正確送達鐵粉；< 15% = 鎖鐵粉圈需廣推
3. **互動率**（互動 / 瀏覽者）> 15% = 優秀、> 25% = 異常好、< 5% = 真差
4. **下游轉化**：GitHub star、Line 入群、訂閱等實際人數（真 KPI）

**4/4 通過** = 成功（即使讚只有 15）。**全部踩下面紅線** = 真 flop。

---

## ⚠️ 真 flop 紅線（必須全部符合）

| 條件 | 門檻 |
|---|---|
| 讚 | < 10 |
| 互動率 | < 5% |
| 追蹤者比例 | < 30%（連鐵粉都不看）|
| 連結點擊 | = 0 |

**全符合才是真 flop**。單一低不代表 flop（如 Day 2 讚 23 但互動率 19.4%、連結點擊 2.35% = 其實很好）。

## 📍 非追蹤者比例監控（突破鐵粉圈）

| 非追蹤者比例 | 判讀 | 行動 |
|---|---|---|
| > 70% | 🔥 演算法廣推中 | 繼續同類內容 |
| 40-70% | 📈 混合觸及 | 正常 |
| 15-40% | 🟡 偏鐵粉 | 可接受 |
| < 15% | ⚠️ **鎖鐵粉圈** | 下一篇必用 F8-F13（見 formulas.md Part 2）|

連續 3 篇 < 15% 非追蹤者 = 演算法已歸類「鐵粉限定創作者」，必須強制擴散。

## 🎭 2 種貼文類型框架（不用同標準比較）

| 維度 | **擴散型**（Day 1 模式） | **深化型**（Day 2 模式） |
|---|---|---|
| 非追蹤者比例 | > 70% | < 40% |
| 讚 | 高 | 低 |
| 互動率 | 中 | **更高** |
| 連結點擊率 | 極低（0.01%）| **極高（2%+）** |
| 實際功能 | 大量路人觸及 | 鐵粉真的 star / fork / 入群 |
| 公式範例 | F6b / F10 / F8 | F4 / F6b 續集 / F13 |

**意義**：擴散型拉新粉、深化型轉化既有粉絲。兩種都是成功，只是功能不同。讚數不可互比。

## 🏆 真正的 KPI：Line 社群 / GitHub star，不是讚

Day 1 實證：374 讚只是表面，**+1,150 Line 社群成員**才是真 ROI。

評估爆款「值不值得」時，問：
- 帶了多少真實社群成員？
- 帶了多少 GitHub star / fork？
- 帶了多少工坊付費學員？
- 帶了多少訂閱 / 追蹤者？

如果這些 downstream 轉化**沒動**，再高的讚也只是數字。

## 📝 排名指標（最近 10 篇相對表現）

FB Insights 會顯示「這篇在最近 10 篇中排名第 N」。使用這個比絕對讚數有意義：

- **排名 1-3**：表現突出，分析原因複製
- **排名 4-6**：正常範圍
- **排名 7-10**：underperform，檢討時段/公式/主題

**原則**：用**排名**評估「這篇對你自己好不好」，用**絕對數**評估「這篇對外部世界強不強」。
