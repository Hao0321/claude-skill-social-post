## Case 1: Day 1 F6b meta-ship mega-viral（2026-04-21 02:13）

**內容**：「Claude 這次真的殺瘋了...這篇貼文就是它自己發的...」

### 72h 後 plateau 數據

| 指標 | 數字 |
|---|---|
| 瀏覽次數 | 125,315 |
| 瀏覽者 | 74,510 |
| 互動次數 | 11,416（率 15.3%） |
| 讚 | 374 |
| 留言 | 452 |
| 分享 | 80 |
| 查看更多點擊 | 10,437 |
| 連結點擊（外部） | 只 7 |
| FB 追蹤者增加 | +167 |
| **Line 社群增加** | **+1,150**（原 ~800 → 1,952） |
| 排名（最近 10 篇） | **1/10** |

### 關鍵觀察

1. **97% 非追蹤者觸及** — 演算法廣推，跨過粉絲圈
2. **外部連結點擊極低（7/74510）** — 讀者不點 GitHub 連結；Line 社群 +1,150 是透過留言 +1 → 作者私訊拉人達成
3. **72h 後仍在漲** — evergreen 長尾，非一次性煙火
4. **Meta AI 副作用**：FB 會把爆款貼文當「Manus AI 範本」在使用者自己的 profile 展示給自己看（僅作者可見），不影響效果

### 複製難度警告

「殺瘋了」「真的太神啦」類 hype 開頭對**非追蹤者**已衰減 40%。F6b 只能**一次性爆破 + 鐵粉收割**，連續用 = 鉤子燒完。想再突破鐵粉圈請用 F8-F13（見 formulas.md）。

1. Day N 正文炫 Day N-1 讚數（try-hard 感）
2. 02:16 凌晨不是一般黃金時段

## Case 3: Day 3 Mode A 真 underperform（2026-04-23 09:48）

**內容**：「寫 skill 寫到天亮，發現一件事：Claude 比咖啡有效☕」

### 數據

| 指標 | 數字 |
|---|---|
| 瀏覽者 | 373（後來 plateau） |
| 讚 | 13 |
| 留言 | 5 |
| 分享 | 0 |
| 互動率 | 11% |
| 追蹤者比例 | 92.9% |
| 連結點擊 | 資料不足 |
| 排名 | **7/10** ⚠️ |

### 錯誤歸因

1. **時段錯**：09:48 早上對 35-44 AI 受眾無效（他們在開會/通勤不滑 FB）。實證對比：02:13=72K、02:16=767、09:48=373
2. **內容太輕**：48 字單句無 hook、無 CTA、無 substance → 演算法看不到 engagement 信號去推
3. **連結點擊 = 0**：沒連結無法轉化

### 新規則：AI/tech 創作者受眾 FB 黃金時段是 **22:00-01:00 夜貓時段**，非傳統 9-11 AM

## Case 4: Day 4 F4 投票「鐵粉限定觸及」（2026-04-24 21:00）

**內容**：「Claude Code 台灣社群這週一口氣進了 1000+ 人，總數來到 1,952...」

| 指標 | 數字 |

| 讚 | 26 |
| 留言 | 5（F4 投票應有 80+ 但未達）|
| 分享 | 0 |
| 互動率 | **28.8%** 🔥（歷史最高）|
| 追蹤者比例 | **93.9%** |
| 非追蹤者 | 只 33 人 |

### 觀察

**連續 Day 2/3/4 都 > 90% 追蹤者觸及** = 演算法已判定「鐵粉限定創作者」，不再嘗試廣推。

F4 投票預期瀏覽者 2-5K 未達成，因為演算法沒把這類 community-focused 內容推給非追蹤者。但**對既有粉絲超級 engaged**（28.8% 互動率）。

### 啟示

Day 5+ 必用 Part 2（F8-F13）廣推公式強制擴散：
- F8 tag 外部實體（@Anthropic）
- F12 時間戳直播（future tense）
- F10 居中開炮（爭議性）
- F9 我錯了（脆弱感）

F4 / F6b / F13 這類社群自我指涉的內容**再用演算法會繼續限縮**。
