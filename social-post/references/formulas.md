# 7 個貼文公式（小帳號可複製）

篩選後的 viral 公式。**過濾掉**：名人/大號（>100K 粉）、付費廣告、MLM/博弈、HubSpot 式泛泛 tips。

選公式看兩件事：(1) 今天的**目標**（社群/工坊/KOL/聯盟）(2) **平台**（Threads 吃短句高回覆、FB 吃長敘事、IG 吃圖、X 吃 thread）。

每個公式給：**骨架 / 例 / 為何有效 / 複製度 1-5 / 主攻平台**。

---

## F1｜Day-N 開發日誌

**骨架**：`Day N 做＜專案＞：今天解決了＜具體問題＞，卡了＜時間＞在＜坑＞。明天要＜下一步＞。`
**例**：Day 6 做 Claude Skill：今天把邀請碼自動派發跑通了，卡了兩小時在 contenteditable 沒焦點。明天接 Threads 自動發文。
**為何**：連載有追劇效應，第 N 篇吃第 1 篇累積的讀者。小號靠累積不靠爆。
**複製度**：5/5
**平台**：Threads（主）、X

---

## F2｜截圖先丟再講

**骨架**：`[大圖/截圖] + 一句話標題 + 2-3 行解釋 + 一句 CTA`
**例**：Claude 跟我說「這段 code 我拒絕寫」我第一次看到 AI 罷工，草。[截圖]
**為何**：演算法看 media dwell time。截圖 = 免費 dwell time；zoom/save 會把你推到 Explore。
**複製度**：5/5
**平台**：FB、Threads、IG

---

## F3｜實測翻車版

**骨架**：`我用＜工具＞＜N＞天，＜正向結果＞。但有三件事沒人告訴你：1)... 2)... 3)...`
**例**：我用 Claude Code 跑社群自動化 30 天，接了 5 個案。但有三件事沒人告訴你：1) skill 一多 context 就爆 2) Chrome MCP 三天 FB 改版一次 3) Max 流量半天見底。
**為何**：純實測已 fatigue，「翻車/沒人說」反轉能撐。save 率約純誇獎文 2 倍。
**複製度**：4/5
**平台**：FB 長文、X thread

---

## F4｜社群里程碑 + 投票

**骨架**：`＜社群＞＜人數＞！！！＜時間敘述＞，沒想到＜情緒＞。留言告訴我想看 A)... B)... C)...，票最多的＜行動＞。`
**例**：Claude Code 破 1200 人！！！才一個月，沒想到。留言想看 A) Skill 開發 B) Chrome MCP C) 社群運營，票最多的週五開工坊。
**為何**：慶祝+選擇一拍兩響。留言權重 > 讚，小號靠留言上分。
**複製度**：5/5
**平台**：FB、Threads

---

## F5｜工具對打

**骨架**：`＜A＞ vs ＜B＞ 跑＜具體任務＞。結論：＜洞見＞。細節：[三行對比]`
**例**：Cursor vs Claude Code 跑 skill 重構。結論：Cursor 改得快、Claude Code 想得深。我早上 Cursor 衝刺、下午 Claude Code 收尾。
**為何**：對比文 save 率 3-5 倍。演算法搜工具名時會 surface 給非追蹤者。
**複製度**：5/5
**平台**：X thread（主）、Threads

---

## F6｜Mode B 爆款（使用者自驗證 100+ 讚，已有兩版實證）

### F6a：邀請碼版（原型）
```
[成果炸場] ＜工具＞＜動詞＞＜具體數字＞＜成果＞！！！你沒看錯！！＜賣點＞！！真的太神啦！！！
[教學引流] 還不會使用的可以去看看我的＜工具＞教學！！
[邀請碼 CTA] 沒有邀請碼的我可以給你好幾組😂
[可選合作] ＜社群/品牌＞跟＜工具＞官方達成合作了🎉🎉🎉
```
**實證**：OiiOii 系列 100+ 讚。

### F6b：meta-ship 變體（2026-04-21 驗證 300+ 讚 / 400+ 留言 / 40+ 分享）
```
[成果炸場 + meta 鉤子] ＜工具/AI＞這次真的殺瘋了，我直接做了一個＜產品＞能＜能力敘述＞！！！你沒看錯！！＜自證懸念，例：這篇貼文就是它自己發的＞！！真的太神啦！！！

[過程數字] 剛剛花＜時間＞讓它＜具體動作＞，＜量化成果，例：13 篇貼文、學會我愛用XX＞

[使用體驗敘事] 現在我說「＜指令＞」它就＜動作＞

[社群召集 CTA（非邀請碼）] ＜未來計畫＞，沒入群的留言我拉你
```

**實證**：2026-04-21 凌晨「skill 自己發」300+/400+/40+（24h 內）。**留言/讚比 > 1.0**，因為 CTA「留言我拉你」觸發大量求拉群留言 = 直接社群增長（+60 追蹤者）。

**為何**：
- **meta 鉤子**（這篇就是它自己發的）比邀請碼更強：引發 curiosity，不需要人家懂工具就想回應
- **留言 CTA 取代邀請碼 CTA**：更乾淨、更導向社群，不會看起來在賣東西
- 演算法看留言權重 > 讚權重；這個版本是「社群成長機器」

**複製度**：5/5
**平台**：FB（主戰場）、IG（配圖時）

### 何時用 F6a vs F6b
- **用 F6a**：推廣**他人的工具**（OiiOii / Runway / 某款 AI）、有邀請碼或 referral 制度
- **用 F6b**：推廣**自己做的東西**、社群里程碑、shipping announcement

## F7｜POV 吐槽（Threads 專屬）

**骨架**：`POV：你是＜身分＞，＜場景＞，然後＜反轉/吐槽點＞。`
**例**：POV：你是 AI 創作者，凌晨三點 Claude 突然說「此對話已達上限」。草。
**為何**：Threads 2026 演算法對 <60 字 + 回覆率 >5% 的短文給最大推送。
**複製度**：4/5
**平台**：Threads
