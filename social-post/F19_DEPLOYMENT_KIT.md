# F19 部署包 — Threads 破圈成長操作手冊（工坊使者版）

> 一份可以直接照做的 Threads 立場宣言公式手冊。給工坊夥伴用，開源公開，自由 fork。
> 規格出處：formulas.md（F19 條目 + 可攜性 proven 子節 + Thread 排版鐵則 + 骨架）／ rules.md（R19 / R24 / R25 / R34 / R41）。

## 一、一句話定位

**F19 = 反賣課的立場宣言，一個可以下放、可以複製的破圈成長公式。**

它的引擎是「反賣課的 tribal 立場」，不是誰的個人聲望，所以內容可以攜帶、可以交給不同帳號各自去發。你不是在賣什麼，你是在表態，讀者轉發你的貼文就是在幫自己站隊。

## 二、proven 實證（匿名化）

F19 已經由多個彼此獨立的素人夥伴帳號驗證過，全部從近乎零基數複製成功：

| 夥伴帳號 | 發文前粉絲（基數區間）| 單篇成果 |
|---|---|---|
| 夥伴 A | 約 100 | → 4,600 粉 / 17.4 萬瀏覽 |
| 夥伴 B | 500-600 區間 | → 9,220 粉（+8,652）/ 53 萬瀏覽 |
| 夥伴 C | 100-200 區間 | → 4,000+ 粉 |
| 夥伴 D | 100-200 區間 | → 4,000+ 粉 |

**判定**：baseline 全部落在 100-600 粉之間，觸及（17.4 萬～53 萬）跟漲粉（單篇 +4,000～8,600）不可能來自帳號自身的既有觸及，主體是演算法對 F19 內容的陌生廣推。「大帳號吃順風」被排除，格式驅動證據確鑿。≥3 個基數各異、彼此獨立的帳號都複製成功，所以判定 **F19 可攜性 = proven**。

**這代表**：你不需要先養大帳號。基數 100 粉也能靠一篇 F19 破圈。這是這套公式最被低估的地方 —— 一個可複製、可下放的破圈公式。

> ⚠️ **fork 者注意**：上面這張表是「別人的戰績」。你發自己的 F19 時，別把任何人的 @handle 或私域絕對數字照抄進正文，只借「公式」不借「別人的隱私」。

## 五、排版鐵則

從真實爆款反向工程出來的硬規格，一條都不能漏：

| 鐵則 | 規格 |
|---|---|
| **段落數** | **1 段，不換行**（寫成多段是 FB 思維誤帶）|
| **句數** | **2-5 句塞在同一段內** |
| **分隔符** | **連續逗號流 +「！」分隔**（不用空行、不用換行）|
| **！級別** | **「！」單個或「！！」最多 — 絕對不用「！！！」** |
| **字數** | **60-150 字**（比 FB 短很多）|
| **emoji** | 可有可無（有些爆款有用、有些沒用，不強求）|
| **連結** | **正文絕不附連結（R25）** |

## 六、keyword 清單

Threads 演算法會偵測熱門 keyword 並主動推播。**一篇至少塞 ≥2 個**：

```
免費 ／ 付費 ／ 韭菜 ／ 賺錢 ／ 幹翻 ／ 智商稅
```

塞得越自然越好，但數量要夠 —— 這是觸發演算法主動推的機制之一。

## 七、voice 鐵則（反 AI 腔）

| 項目 | 規則 |
|---|---|
| **標點** | 日常內文只用 `，`（逗號）`、`（頓號）`。`（句號）。**但 F19 的 hype 照公式用「！」／「！！」** — 這是真實爆款驗證過的 proven 裝置，不是 AI 腔 |
| **禁用詞（B 類黑名單）** | 護城河 / 本質 / 真正的 X / 賦能 / 複利 / 底層邏輯 / 降維 / 沉澱 / 佈局（出現即重寫）|
| **真實感來源** | 具體 > 抽象，講真的事、真數字、真動作；口語詞（弄、搞、跑、改、想通、有夠、扯、爽）；自嘲 > 宣言 |
| **TM 級用詞** | F19 可用（增加真實感，演算法判 organic creator 而非行銷帳號）|

「語氣強度」跟「反 AI 腔」是兩個獨立的軸 —— 你可以又 hype 又不像 AI。真正像 AI 的是**抽象空詞 + 過度工整**，不是「！」。

## 八、🔑 拉爆轉發的關鍵升級：stance 綁 artifact

**這是把轉發從幾十衝到上千的核心槓桿。**

抽象喊口號（「我免費分享技術」）有天花板。要突破天花板，把反賣課的 stance **綁在一個「當下超火、超具體、路人一看就想要」的 artifact 上**。

- 抽象 stance 版：觸及 26 萬 / 轉發 76（有天花板）
- stance 綁具體當紅 artifact 版：觸及 44 萬 / 轉發 1,197（前高的 15.7 倍）

**striking 具體細節**（例如「100 個剪輯記憶」這種數字）比抽象「我公開技術」勾人 10 倍（R41）。demo > claim 在 Threads 一樣成立（R19①）。

**操作**：你手上如果有一個剛做好、正在紅、大家看了會想要的東西，就把它綁進 F19 的 giveaway 位置，別只喊抽象口號。

## 九、🔑 拉大觸及的第二槓桿：連發累積 momentum（R24）

一篇孤立的 F19 天花板大約 50K。**≤ 5 天內連發 2-3 篇**同結構 F19，會累積 viral momentum，衝到 100K-150K+。

- 結構可以重複（同一套五元素骨架、同排版鐵則）
- 但**每篇的敵人 / 題材要換**（別連三篇罵同一個對象，會膩）
- 節奏：一篇 → 隔 2-4 天 → 下一篇，趁前一篇餘熱還在

這是 Thread 特有的（Threads 演算法 burnout 比 FB 慢，可以較頻繁）。

## 十、讀自己的成績：Thread 三指標分軸

Threads 的三個指標要**分開讀，不要混為一談**（R19④）：

| 指標 | 意義 | 深度 |
|---|---|---|
| **轉發（repost）** | 高承諾的 identity 表態，衡量「多少人願意把你的立場原封不動掛自己名下」= 站隊深度。Thread 最深信號，比讚強 5-10 倍 | 最深 |
| **分享（share）** | 低承諾、高擴散引擎（不掛名，只傳出去），把內容帶進陌生 feed | 中 |
| **瀏覽** | 最淺指標，由 keyword 主動推 + 分享擴散 + 站隊情緒共同拉動 | 最淺 |

**診斷讀法**：

- **高瀏覽 + 低轉發率** = 你綁 artifact 綁得不夠，擴散力有了但表態深度淺（缺具體 artifact 錨點）
- **轉發率高** = 站隊深、identity 強

要拉深度看**轉發率**，不是看瀏覽。**高瀏覽不等於高轉發。** 參考基準：F19 純血的 per-view 轉化率做到 0.60% 就是帳號歷史最高等級（R23）。

## 十一、giveaway 反摩擦

- 公開連結**放留言區**，不放正文（正文外連 = 降權，R25）
- ⚠️ **Thread 的留言區連結要慎用**（單則就好、別洗）；FB 留言區才是乾淨的 ✅（R25）。正文永遠不外連。
- 姿態：「拿去用就好，順手點顆星」
- **不強制留言關鍵詞**（Thread 的 CTA 軸是轉發／identity 表態，不是留言索取，keyword 留言 CTA 在 Thread 反而無效，R19）

## 十二、常見錯誤

| 錯誤 | 後果 |
|---|---|
| 抽象口號、沒綁具體 artifact | 轉發低，撞天花板 |
| 正文放連結 | 觸及降權 30-50%（Thread 唯一的硬降權錯誤）|
| 「！！！」用太多 | 破壞 F19 節奏（最多「！！」）|
| 把 F19 搬去 FB 發 | 被判 spam die |
| 寫成多段換行 | FB 思維誤帶，F19 要 1 段不換行 |

## 十三、⚠️ 誠實射程（別當萬靈丹）

目前所有夥伴案例**共用同一個題材（反付費課程）+ 同一個話題波**（近期的反賣課熱潮）。

所以 proven 的範圍 = **「溫熱話題波內可攜」**。

- ✅ 已證：話題正熱時，反賣課題材，多個素人帳號各自複製成功
- ❓ 未證：冷話題 / 離開這波造浪之後，能不能一樣有效 —— 這是**唯一的開放測試**

**別 over-claim「F19 任何題材都可攜」。** 話題波熱的時候用，別當萬靈丹硬套冷題材。

## 十四、兩個 worked example（，、為主，F19 hype 照公式用 ！／！！）

### 範例 A：綁 artifact 的高轉發版（示意重構 — 依骨架 + 真實戰績 44 萬 / 轉發 1,197 還原，非原文逐字）

> 最近賣課的都在收你錢教你剪片，我做了一套自動剪輯還塞了 100 個剪輯記憶，全部免費送，就 TM 做真的！我就持續開源幹翻所有付費課程！！對我來說這只是興趣而已，送出去對我沒差

**為什麼強**：stance（幹翻付費課程）綁了一個具體、當紅、路人一看就想要的 artifact（自動剪輯 + 100 個剪輯記憶）。路人看得到「這東西幫我做什麼」，不是空喊口號 → 轉發暴衝。

### 範例 B：抽象版（示意 — 立場對但沒綁 artifact，26 萬瀏覽 / 轉發只有 76）

> 我不靠 AI 課程賺錢，我會把技術經驗全部公開分享，目前 2 到 3 天更新一次教學！！來幹掉所有付費課程

**為什麼有天花板**：立場對、keyword 有、五元素齊，瀏覽也很高（26 萬），但**只喊抽象立場「我公開技術」，沒綁一個具體看得到的東西** → 路人有情緒但沒有錨點可以站隊，轉發卡在 76。

### 差在哪（一句話）

**兩篇立場一樣、瀏覽都很高。差別只在 A 綁了一個「路人一看就想要的具體 artifact」，B 只喊抽象口號。** 這一個變數，就是轉發 76 → 1,197（15.7 倍）的分水嶺。抽象 stance 有天花板，綁具體 artifact 破天花板。
